# vendmat-excel-analysis
This project focuses on analyzing a food ordering dataset (Vendmat) using Excel to derive business insights related to sales performance, customer behavior, and operational trends.

The analysis includes key metrics such as total orders, revenue, average order value (AOV), and customer ratings. It also explores patterns across different dimensions like tech parks, shifts, and days of the week.

Key highlights of the project:
- Identified top-performing tech parks based on revenue and orders
- Analyzed peak demand periods (Lunch, Friday trends)
- Evaluated top-selling and high-revenue items
- Measured revenue contribution (%) by each tech park
- Identified repeat customer behavior and its impact
- Performed data cleaning and validation in Excel

This project demonstrates strong data analysis skills using Excel, including data cleaning, pivot tables, aggregations, and business insight generation.
